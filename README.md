# 🚀 VisualX-Lab

https://visual-x-lab.vercel.app/

### *Visualize. Transform. Understand.*

An interactive web-based Computer Graphics simulator designed to demonstrate **2D and 3D transformations** in real-time.

---

## 📌 Project Overview

VisualX-Lab is a browser-based graphical tool developed as part of the **Computer Graphics and Animation (21CSE255T)** course.

It allows users to visually explore and understand core concepts of:

* 2D Transformations (Unit 2)
* 3D Transformations (Unit 3)

The application provides an intuitive interface where users can apply transformations and instantly observe their effects.

---

## 🎯 Features

### 🔷 2D Transformations

* Translation (X, Y movement)
* Rotation (angle-based)
* Scaling (uniform & non-uniform)
* Reflection (X-axis, Y-axis)
* Shearing
* Composite transformations
* Matrix representation display
* Before vs After visualization

---

### 🔷 Clipping Algorithms

* Cohen-Sutherland Line Clipping
* Interactive clipping window
* Highlighted visible & clipped segments

---

### 🔷 3D Transformations

* 3D Object (Cube/Wireframe)
* Translation (X, Y, Z)
* Rotation (X, Y, Z)
* Scaling
* Axis visualization (X, Y, Z)

---

### 🔷 Projection & Viewing

* Perspective Projection
* Orthographic Projection
* Real-time switching between projections
* Depth perception visualization

---

## 🧠 Learning Objectives

* Understand geometric transformations using matrices
* Visualize real-time transformation effects
* Learn projection techniques used in graphics systems
* Translate theoretical concepts into interactive simulations

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glassmorphism + Neon UI)
* JavaScript (ES6)
* Canvas API (2D rendering)
* Three.js (3D rendering)
